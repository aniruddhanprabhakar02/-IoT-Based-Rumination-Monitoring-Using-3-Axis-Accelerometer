# Hardware Design Files

This directory contains the KiCad design files for the custom PCB developed for the IoT-Based Cattle Rumination Monitoring System.

## Project Files

| File | Description |
|------|-------------|
| `rumination.kicad_pro` | KiCad project file |
| `rumination.kicad_sch` | Schematic design |
| `rumination.kicad_pcb` | PCB layout |

## Gerber Files

The `gerber_files/` directory contains the fabrication-ready Gerber files generated from KiCad.

> **Note:** The PCB was designed as a fabrication-ready prototype. The experimental validation presented in this repository was performed using a perfboard-based hardware prototype rather than the fabricated PCB.
